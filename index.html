About the Project

Independence,
powered by AI.

SIGHT acts as a second pair of eyes — translating the visual world into clear voice guidance, in real time. It is designed around one goal: giving blind users the confidence to act on their own.

Identify banknotes instantly to handle money safely
Recognize familiar faces of friends and family
Read printed text aloud from any document or label
Detect obstacles on the path to move around safely
Ask open questions about surroundings using AI
Offline-First Design

Works without internet.

Core accessibility features — currency detection, text reading, face recognition and obstacle detection — run entirely on the device. SIGHT keeps working in areas with weak or no connection, where users need it the most. Cloud intelligence is used only to extend capability, never as a requirement.

How It Works

From camera to clear guidance.

A layered pipeline keeps responses fast and private — most processing never leaves the phone.

01

Camera / Voice Input

The user triggers SIGHT by voice and points the camera.

02

On-Device AI Processing

TensorFlow Lite & MediaPipe run detection locally — fast and offline.

03

Gemini Cloud AI

Optional cloud layer for complex reasoning about a scene.

04

Voice Feedback to User

Results are spoken back in clear, natural language.
